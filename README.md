Laser Engraving – Hobby Website
This project is a simple website focused on hobby laser engraving and cutting. 
The main goal of the website is to present a personal project, explain basic laser technology, and show practical use 
of engraving on different materials.
The website consists of several pages with clear navigation and a unified design. 
One of the main features is a materials table, which is dynamically loaded from a JSON file using JavaScript.

Website Structure

Home – introduction to laser engraving
About Me – personal information about the author and the hobby
Technology – basic explanation of how laser technology works
Gallery – examples of finished products (lightbox gallery)
Materials – table of materials and laser settings loaded from JSON
Order – order form with basic validation
